Designing Graphical Display System with Large LCD for PLC Industrial

In this event, we will discuss different types of  Graphics Display available in the market, various technology involved and basic operation of Graphics Display. Hardware and software design consideration will also be discussed in this event. This event will be useful for those who want to get a quick start on GUI based application. Here demonstration will be give on Imbuent’s Graphics Display Board.


Embedded Linux on ARM
Linux running of Embedded System is Embedded Linux. On ARM processors which has MMU we can run Embedded Linux is the use of Linux Operating System in embedded computer systems such as mobile phones, personnel digital assistants, media players and other consumer electronics  devices, networking equipments, navigation equipments and medical equipments. According to the survey conducted by Venture Development Corporation, Linux was used in 18% of embedded engineers. In this we will show various demos of Embedded Linux on ARM series MPU.
